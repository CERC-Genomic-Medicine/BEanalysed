BEanalysed — In-Browser Ports

Static HTML/JavaScript ports of the Python scripts in BEanalysed_2/Scripts/. Every page reads data locally in the browser (no server), and regenerates plots automatically as inputs change.

ROC

BE_ROC.py — cumulative rank plots and combined ROC curves from a consolidated MaGeCK xlsx.

ROC per Gene

BE_ROC_PerGene.py — ROC curves split by gene and by positive-consequence category.

Lollipop

BEanalyzed_lollipop_plot.py — lollipop plot of LFC vs protein position with feature annotations and biological thresholds.

Scatter

scatter_plot_BEanalysed.py — pairwise scatter of two sheets by shared id, with Pearson r.

Repeat on Repeat

RepeatOnRepeat.py — pairwise scatter of repeat columns across multiple TSV files.

ROC Quantile Sweep

test_ROC_perf.py — AUC as a function of the negative control quantile window, with LOWESS smoothing.

3D Structure Viewer

In-browser port of chimeraX.sh / chimeraX_clusters.sh using 3Dmol.js. Load a PDB/mmCIF plus one or more .defattr files and colour residues dynamically with your choice of palette, representation, and highlight mode.
